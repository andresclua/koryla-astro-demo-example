# Koryla Astro Demo

Live A/B testing examples using Koryla edge functions on Netlify.

- `/` — URL rewrite experiment (homepage vs variant-b)
- `/pricing` — pricing page experiment
- `/sdk-demo` — component-level testing with `@koryla/astro`

[Live demo →](https://astro-demo.koryla.com) · [Docs →](https://koryla.com/docs/sdk-astro)

## Setup

```bash
cp .env.example .env
# add your KORYLA_API_KEY and KORYLA_API_URL
npm install
npm run dev
```
